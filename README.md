<h1>Python - Image-Filtration Mini Project</h1>


<h2>Description</h2>
It`s a Image-Filtration project to edit and apply filters on photos using several techniques like:

- Contrast 
- Crop
- Gamma Filter
- Gaussian Filter
- Histogram
- Median Filter
- Negative Filter
- Remove Noise
- add Salt&Pepper niose
- Sharpen Filter
